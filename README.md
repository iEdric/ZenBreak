# ZenBreak - 办公室放松工具

> "休息是为了走更长远的路"
> 一个专为办公室工作者设计的短暂放松应用

## 📖 项目简介

ZenBreak（禅境时刻）是一个专为办公室工作者设计的短暂放松工具，帮助你在繁忙的工作中获得片刻宁静。通过科学的放松指导和美观的可视化界面，让你快速重拾专注力和创造力。

## ✨ 主要功能

### 🧘 核心功能

- **三种放松模式** - 深呼吸、眼部放松、片刻冥想
- **智能计时器** - 可自定义1/3/5分钟放松时长
- **引导式放松** - 专业的放松指导和步骤说明
- **美观可视化** - 动画效果配合呼吸节奏和眼部运动
- **响应式设计** - 完美适配桌面和移动设备

### 📊 功能特性

- 实时倒计时显示
- 自动循环的放松指导
- 平滑的动画过渡效果
- 现代化的用户界面设计
- 中文界面，贴心本地化

## 🛠️ 技术栈

- **前端框架**: React 19.2.3
- **开发语言**: TypeScript 5.8.2
- **构建工具**: Vite 6.2.0
- **样式方案**: Tailwind CSS (CDN)
- **图标库**: Lucide React
- **字体**: Noto Sans SC (中文字体)
- **动画**: CSS动画和Tailwind动画类

## 💡 使用说明

### 1. 开始使用
   - 打开应用，选择你需要的放松方式
   - 设置合适的放松时长（1/3/5分钟）
   - 点击"开始放松"按钮进入放松模式

### 2. 三种放松模式

   **深呼吸模式** 🫁
   - 跟随屏幕上的呼吸节奏（吸气4秒-保持4秒-呼气4秒）
   - 可视化动画会随着呼吸节奏变化
   - 专业的呼吸指导文字

   **眼部放松模式** 👁️
   - 每3秒切换一次眼部运动指令
   - 包括向上看、向下看、向左看、向右看和闭眼放松
   - 缓解长时间屏幕使用带来的眼部疲劳

   **片刻冥想模式** 🧘
   - 流动的彩色圆形动画
   - 营造宁静的冥想氛围
   - 适合短暂的思维清空

### 3. 放松流程
   - **准备阶段** - 选择模式和时长，点击开始
   - **放松阶段** - 跟随可视化和文字指导进行放松
   - **完成阶段** - 显示完成信息，可选择再来一次或返回主页

### 4. 界面说明
   - **主页** - 模式选择和时长设置
   - **放松页面** - 显示倒计时、当前指导和可视化
   - **完成页面** - 显示放松成果和操作按钮
